# 网络模型

### 网络

一个容器的集合，同一个网络上的容器可以通过这个网络互相通信，直接通过 `bridge` 实现设备网络互联

### 网络端点

用于连接容器和网络，保证容器内部和网络的通信，一段在容器内一段在网桥
端点的信息传递需要依靠网络驱动和 `IPAM` 配合使用

### 网络驱动

网络功能组件，不同的驱动对网络的创建，连接和销毁的策略不同
创建网络时完成网络初始化
容器启动时配置网络端点

### IPAM

用于网络 `IP` 地址的分配和释放，包括容器 `IP` 地址和网络网关 `IP` 地址
将传入的 `IP` 网段去分配一个可用的 `IP` 地址给容器和网关
